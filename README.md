# internship
Kodi i projektit selenium te intershipit.

Per t'i ber run projektit duhet qe te ndiqni hapat e meposhtem:
  1. Git clone projektit.
  2. Beni run ne maven komanden 'mvn clean install'
  3. Ne fund per te ber run test caset duhet qe te ndryshoni ne klasen GlobalVariables.EMAIL me vleren qe ju deshironi.
  4. Beni run komanden 'mvn test'
  5. Ne folderin target/surefire-reports/Projekt/Comercial Test.html do te gjeni raportin e gjenerua dhe screenshot ne rast fail te test caseve.
